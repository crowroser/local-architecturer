# analyze_project MCP Aracını Güncelleme Planı

## Sorun
`analyze_project` MCP aracı şu an sadece `scanner.scan()` sonucunu döndürüyor:
- packages
- dockerConfigs
- dependencies

Yeni eklenen 10 özellik (database schemas, proxy configs, dataflow, security boundaries, AI profile, hardware devices, time-travel, CI/CD pipelines) bu araçta görünmüyor.

## Çözüm
`analyze_project` aracını genişleterek tüm yeni verileri tek bir yanıtta döndürmek.

## Değişiklik
Dosya: `src/mcp/server.ts`

Mevcut:
```typescript
const result = await scanner.scan();
return { content: [{ type: 'text', text: JSON.stringify(result, null, 2) }] };
```

Yeni:
```typescript
const [result, dbSchemas, proxyConfigs, dataFlows] = await Promise.all([
  scanner.scan(),
  scanner.getDBSchemas(),
  scanner.getProxyConfigs(),
  scanner.getDataFlows(),
]);

// Security boundaries
const analyzer = new SecurityBoundaryAnalyzer();
const allServices = result.dockerConfigs.flatMap(c => c.serviceDetails || []);
const securityBoundaries = analyzer.analyze(result.dockerConfigs);

// AI Profile
const profiler = new AIProfiler();
const aiProfile = profiler.profile(allServices);

// Hardware devices
const hardwareNodes = result.dependencies.nodes.filter(n => n.type === 'hardware');

const fullResult = {
  ...result,
  databaseSchemas: dbSchemas,
  proxyConfigs,
  dataFlows,
  securityBoundaries,
  aiProfile,
  hardwareDevices: hardwareNodes,
};

return { content: [{ type: 'text', text: JSON.stringify(fullResult, null, 2) }] };
```
