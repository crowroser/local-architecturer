# Sürüm Yükseltme ve Push Planı

## Değişiklik
- `package.json`'daki `"version": "0.4.1"` → `"0.5.0"` olarak değiştir
- Versiyon zaten `getVersion()` ile package.json'dan okunuyor, başka yerde hardcoded yok

## Doğrulama
1. `npm run build` — başarılı derleme
2. `npm test` — tüm testler geçmeli

## Git
- Değişiklikleri commit et
- Remote'a push et
